# Blackjack Pro

A sleek, luxury-themed Blackjack game built with React, TypeScript, and Tailwind CSS.

https://blackjackpro-oz.vercel.app

## Features

- Full Blackjack gameplay: Hit, Stand, Double Down, Split
- Realistic dealer AI
- Chip-based betting system
- Smooth card animations with Framer Motion
- Responsive design

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- Vite

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
